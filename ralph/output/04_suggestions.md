## Custom Mapping Implementation Examples

### Move Lines with Alt+j/k

```lua
vim.keymap.set("n", "<A-j>", ":m .+1<CR>==", { desc = "Move line down" })
vim.keymap.set("n", "<A-k>", ":m .-2<CR>==", { desc = "Move line up" })
vim.keymap.set("v", "<A-j>", ":m '>+1<CR>gv=gv", { desc = "Move selection down" })
vim.keymap.set("v", "<A-k>", ":m '<-2<CR>gv=gv", { desc = "Move selection up" })
```

### Quick Save All

```lua
vim.keymap.set("n", "<A-s>", ":wa<CR>", { desc = "Save all buffers" })
```

### Toggle Terminal

```lua
vim.keymap.set("n", "<F6>", ":ToggleTerm<CR>", { desc = "Toggle terminal" })
vim.keymap.set("t", "<F6>", "<C-\><C-n>:ToggleTerm<CR>", { desc = "Toggle terminal" })
```
