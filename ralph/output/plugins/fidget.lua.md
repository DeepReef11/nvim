# Plugin: fidget.nvim

**File:** `lua/plugins/ui/fidget.lua`

## Overview

Fidget.nvim is a UI plugin that shows LSP progress indicators in the corner of the editor.

## Custom Keybindings

**No custom keybindings defined in this plugin configuration.**

The plugin uses default settings with empty `opts = {}`.

## Notes

- Fidget.nvim typically doesn't require keybindings as it displays information passively
- The plugin shows notifications for LSP progress automatically
