## Keybindings

This plugin is **disabled** but has the following keybindings configured in the suggestion keymap:

| Key | Mode | Action | Description |
|-----|------|--------|-------------|
| `<TAB>` | Suggestion | accept | Accept Copilot suggestion |
| `<C-w>` | Suggestion | accept_word | Accept word from suggestion |
| `<C-l>` | Suggestion | accept_line | Accept line from suggestion |
| `<C-j>` | Suggestion | next | Next suggestion |
| `<C-k>` | Suggestion | prev | Previous suggestion |
| `<ESC>` | Suggestion | dismiss | Dismiss suggestion |
