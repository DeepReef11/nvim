# Plugin File: lua/plugins/ui/numb.lua

## Plugin: numb.nvim
- **Repository**: nacro90/numb.nvim
- **Description**: Neovim plugin that peeks lines of the buffer in non-obtrusive way

## Custom Keybindings Defined

*No custom keybindings defined in this configuration file.*

This plugin is loaded with default options (`opts = {}`). The plugin works automatically when typing `:` followed by a line number - it will preview that line without committing the jump.

## Notes
- Plugin is lazy-loaded: `false` (loads immediately)
- Uses default plugin keybindings only
