## Notes

This plugin provides a simple statusline/winbar component showing LSP context (breadcrumbs). It automatically attaches to LSP servers and displays the code context. No custom keybindings are configured.

## Configuration Summary

- Events: CursorMoved, BufWinEnter, BufFilePost
- LSP auto-attach: enabled
- Dependencies: nvim-lspconfig
