# stay-in-place.nvim Keybindings

**Plugin:** [gbprod/stay-in-place.nvim](https://github.com/gbprod/stay-in-place.nvim)

**Description:** Prevents the cursor from moving when using shift and filter actions.

## Custom Keybindings

*No custom keybindings defined - plugin uses default configuration.*

## Notes

- Plugin loaded with `config = true` (default setup)
- Plugin is always loaded (`lazy = false`)
- This plugin modifies the behavior of built-in operators like `>`, `<`, `=`, and `!` to keep cursor position stable
