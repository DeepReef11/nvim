# Plugin: markdown-preview.nvim

**Source file:** `lua/plugins/markdown-preview.lua`

## Plugin Info
- **Repository:** iamcco/markdown-preview.nvim
- **Filetypes:** markdown

## Custom Keybindings

No custom keybindings defined in this configuration file.

The plugin is loaded lazily for markdown files only.

## Notes

This plugin provides the following default commands (no custom keymaps configured):
- `:MarkdownPreview` - Start markdown preview
- `:MarkdownPreviewStop` - Stop markdown preview
- `:MarkdownPreviewToggle` - Toggle markdown preview
